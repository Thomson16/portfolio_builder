# Portfolio Builder Platform

## Overview

Portfolio Builder Platform is a full-stack web application that allows users to create, manage, and publish professional online portfolios without coding. Users can showcase their skills, education, projects, experience, and social profiles through a personalized portfolio page.

## Features

* User Registration and Login
* Secure Authentication
* Create and Manage Portfolio
* Profile Photo Upload
* Portfolio Preview
* Public Portfolio Gallery
* GitHub and LinkedIn Integration
* Responsive Design
* SQLite Database Storage

## Installation

1. Clone the repository

```bash
git clone https://github.com/Thomson16/portfolio_builder.git
```

2. Open project folder

```bash
cd portfolio_builder
```

3. Install dependencies

```bash
npm install
```

4. Run the application

```bash
npm run dev
```

or

```bash
node app.js
```

5. Open browser

```text
http://localhost:3000
```
